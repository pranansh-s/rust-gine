# Rust OpenGL 2D Game Engine

A modern 2D game engine built with Rust and OpenGL, featuring a functional camera system, entity-component architecture, DOM-inspired UI, and resource management.

## Features

- 🎮 **Entity-Component System** for game objects
- 📷 **Functional Camera** with pan/zoom controls
- ✨ **OpenGL 3.3+ Rendering Pipeline**
- 🧩 **Resource Manager** for textures/shaders/assets
- 🖌️ **DOM-inspired UI System** with flexbox layout
- 🔄 **Transform Hierarchy** (position/rotation/scale)
- 📦 **Texture and Sprite Management**
- 📊 **Logging** with sysinfo
